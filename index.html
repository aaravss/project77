About the Console

Visual Studio is an IDE that anyone can use to code. To use VS, you have to first create an empty folder. Then, open Visual Studio and click on "File" in the top-left corner. From there click on "Open Folder" and select the folder you created. After that, click on the open file icon on the folder and type "index.html". This will give you the file needed for coding HTML, and you can start coding!

A console traditionally refers to a computer terminal where a user may input commands and view output such as the results of inputted commands or status messages from the computer, such as errors.


Go ahead! Open the console now by simply right clicking anywhere on the webpage and selecting "Inspect"!
